## Frontend Mentor - Tic Tac Toe Solution
This is my functional solution to the Tic Tac Toe challenge on Frontend Mentor. While the visual assets and design system were provided by Frontend Mentor, all application logic, state management, and styling were implemented by me from scratch.


Overview
The Challenge
The goal was to build a fully functional Tic Tac Toe game that handles complex state transitions. Users should be able to:

View the optimal layout for the game depending on their device's screen size.

See hover states for all interactive elements.

Play the game solo against a "clever" CPU or multiplayer against another person locally.

Bonus: Maintain the game state via browser storage so progress isn't lost on refresh.
